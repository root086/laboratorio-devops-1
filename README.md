Este projeto consiste na implementação de um pipeline manual para deploy de uma aplicação web estática containerizada. 

Link da documentação: https://lucasvitor.vercel.app/#laboratorio

A aplicação foi empacotada utilizando **Nginx** e implantada em uma instância **EC2**, com as imagens gerenciadas via **Amazon ECR**.
